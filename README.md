## Multiplicar console app

Esta es una aplicaciÃ³n para crear una lista de tareas

Ejecutar

```
npm install
```

Para crear una tarea

```
node create -c string
```

Para listar las tareas

```
node app list
```

Para marcar como realizada una tarea

```
node app -d string
```

Para eliminar una tarea

```
node app -e string
```
